# PyLogger
This is a simple library to log your system

## Functions

* getLogSize():Int
* clearLog()
* getLog():str[]
* printLog() # prints to std
* log(text:str, [error:Exception])

## Setup

Open the file and edit the first variable.

```python
LOG_FILE = "put/four/filepath.here"

```
